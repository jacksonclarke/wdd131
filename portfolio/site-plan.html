Overview

Purpose

A handcrafted portfolio site built with HTML, CSS, and vanilla JavaScript, featuring dynamic filtering, animations, and interactive components. Designed with a custom dark theme, original SVG graphics, and a deliberate typographic system to present engineering and cybersecurity project work to prospective employers.

Audience

The target audience for this website is potential employers and clients interested in my software development skills.

Dynamic elements

This website will include dynamic elements such as a responsive navigation menu, interactive portfolio items, dynamic portfolio updates, and a contact form.
